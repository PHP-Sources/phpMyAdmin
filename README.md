# phpMyAdmin

<img src="https://github.com/PHP-Sources/.github/raw/main/logos/phpmyadmin.webp" width="48" height="48" alt="phpMyAdmin">

**phpMyAdmin 5.2.3** – L’outil open source le plus populaire pour administrer les bases de données MySQL / MariaDB via une interface web.

### Points forts
- Interface web intuitive et complète pour gérer vos bases de données
- Support complet de MySQL et MariaDB
- Import / Export de données (SQL, CSV, XML, etc.)
- Gestion avancée des utilisateurs, privilèges et droits
- Outils de recherche, optimisation et maintenance
- Très utilisé par les développeurs et administrateurs système
